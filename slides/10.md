# Metodologia

Para finalizar o ajuste do modelo proposto aos dados da pandemia os autores fizeram:

* Normalização das variáveis \\(S, I, R\\) em \\([0, 1]\\) escalonadas por uma função corretiva \\(f\\) para atingir os números reais de casos / óbitos;
* Separação de recuperados e óbitos à partir do conjunto de indivíduos removidos \\(R\\)
  * \\(D(t) = D_{0} (1 - e^{-k R}) \\) - função não linear que representa o número de óbitos no tempo \\(t\\);
  * \\(R_{c}(t) = R(t) - D(t)\\) número de recuperados no tempo \\(t\\);
* Ajuste dos parâmetros guiados pelos dados de cada país / estados, ajustados empiricamente para cada localidade a partir das curvas obtidas dos dados oficiais;
